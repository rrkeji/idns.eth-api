## 数据版本的比较

### 获取数据库的当前schema

获取所有的表
```
-- type name tbl_name rootpage sql
SELECT * FROM sqlite_master WHERE type='table' ORDER BY name;
-- 
SELECT * FROM sqlite_master WHERE 1 = 1 ORDER BY name;
```
获取表的所有的列
```
-- cid name type notnull dflt_value pk
PRAGMA  table_info("person")
```
